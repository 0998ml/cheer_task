# CheerTASK チア―タスク

## サイト概要
### サイトテーマ
チーム開発の進捗をカンバン形式で可視化し、タスク完了時にメンバー同士で称賛（リアクション）を送り合える、コミュニケーションに特化したタスク管理ツールです。
​
### テーマを選んだ理由
始めてのチーム開発を経験して、進捗が不透明になりがちであることを感じました。
初めてチーム開発に取り組む人がチームのコミュニケーションを楽にとれるように、このようなポートフォリオを制作してみようと考えました
​
### ターゲットユーザ
初めてチーム開発に挑戦するプログラミング学習中のチーム

メンバー間のコミュニケーションを活性化させたいのプロジェクトメンバー
​
### 主な利用シーン
毎日の進捗確認で、誰がどの機能を担当しているか一目で把握したい時

メンバーが大きな機能を実装し終えた際、感謝や称賛の気持ちを気軽に伝えたい時
​
