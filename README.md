## サイト概要
### サイトテーマ
チーム開発の進捗を可視化し、メンバー同士で称賛し合える、コミュニケーションに特化したサイトです。
​
### テーマを選んだ理由
初めてチーム開発を経験した際、メンバー間の進捗状況が不透明になりやすく、情報共有に課題があると感じました。この経験から、初めてチーム開発に取り組む方でも、迷うことなくスムーズにコミュニケーションが取れる環境を提供したいと考え、本サービスを制作いたしました。
​
### ターゲットユーザ
-初めてチーム開発に挑戦する人
-メンバー間のコミュニケーションを活性化させたい人
​
### 主な利用シーン
-毎日の進捗確認で、誰がどの機能を担当しているか一目で把握したい時
-メンバーが大きな機能を実装し終えた際、感謝や称賛の気持ちを気軽に伝えたい時
​
